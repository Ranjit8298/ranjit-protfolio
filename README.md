## 🛠️ Technology Stack
This portfolio is built using core web technologies to ensure maximum performance and zero dependency bloat:

*   **HTML5:** Semantic markup for structure and accessibility.
*   **CSS3:** Custom styling utilizing CSS Variables for consistent theming (`--primary-color`, fast UI transitions, and modular layout using Flexbox and CSS Grid.
*   **Vanilla JavaScript:** Lightweight DOM manipulation for sticky navigation and smooth scrolling mechanics.
*   **Assets:**
    *   **Google Fonts (**`Roboto`**):** Clean and legible typography.
    *   **FontAwesome (v6):** Scalable vector icons.

## ✨ Key Features

### 🖥️ Responsive & Modern UI
*   **Mobile-First Approach:** Flawless rendering on smartphones, tablets, and desktop environments.
*   **Glassmorphism Navigation:** A sticky navbar that dynamically blurs the background on scroll (`backdrop-filter: blur(10px)`).
*   **Micro-Interactions:** Subtle hover animations on cards, buttons, and icons to elevate the user experience.

## ⚙️ Running Locally
To view or modify this project on your local machine, no complex build steps or node packages are required:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Ranjit8298/portfolio.git
    cd portfolio
    ```

2.  **Run it:**
    *   **Option A:** Simply double-click the `index.html` file to open it in your default browser.
    *   **Option B (Recommended):** Use a local development server like VS Code's **Live Server** extension to handle assets perfectly.
        ```bash
        # If you have Python installed, you can quickly spin up a server:
        python -m http.server 8000
        # Then visit http://localhost:8000 in your browser
        ```
